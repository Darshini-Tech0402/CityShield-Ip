# Hybrid Surveillance and Cyber Security Management System

CityShield is a smart city monitoring platform that combines physical surveillance and cyber threat detection.

## Features
• Smart City Dashboard  
• Incident Reporting System  
• Zone Based Monitoring  
• CCTV Live Monitoring  
• Cyber Attack Alerts  
• Google Maps Incident Visualization  

## Tech Stack
Frontend: React  
Backend: Java Servlets (Tomcat)  
Database: MySQL  
Maps: Google Maps API  

## Modules
Dashboard – City incident statistics  
Incidents – Report and manage incidents  
Devices – Surveillance devices monitoring  
City Map – Incident visualization on map  
CCTV – Live monitoring feeds  
Cyber Alerts – Cybersecurity threat alerts
